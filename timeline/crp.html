  4. Parse the CSS into CSSOM
  5. Interpret the js and execeute to complete the HTML parsing
  6. Construct Render Tree combining DOM & CSSOM
  7. Render Tree Layout
  8. Render Tree Paint convertin the visual elements into pixels on the screen
